Highlighter Support

A simple browser extension that highlights text and allows quick annotations.

Help Guide Sponsor View Repository

What it does

Highlight selected text on any web page and save those highlights for later reference without extra clutter.

  • Highlight selected text instantly
  • Attach quick annotations
  • Lightweight and easy to use

Get started

  1. Install the extension from the Chrome extensions panel (Developer mode).
  2. Select text on any page and choose “Highlight”.
  3. Open the extension popup to review or annotate highlights.
